.. py:class:: SearchResult

   Result from a vector search operation.

   .. attribute:: id

      Unique identifier of the vector

   .. attribute:: score

      Similarity score (higher = more similar)

   .. attribute:: text

      Original text associated with the vector

   .. attribute:: metadata

      Additional metadata stored with the vector

   .. attribute:: vector

      Optional embedding vector (not always returned)

   .. admonition:: Example

      >>> result = SearchResult(
      ...     id="doc_123",
      ...     score=0.95,
      ...     text="Installation guide for Python",
      ...     metadata={"source": "manual", "page": 5}
      ... )
      >>> print(f"Found: {result.text} (score: {result.score:.2f})")
