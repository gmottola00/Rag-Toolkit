.. py:function:: normalize_bytes(raw_bytes, fallback_encoding = 'utf-8')

   Normalize bytes into a cleaned UTF-8 string.

   The function attempts charset detection via ``charset-normalizer`` and
   falls back to the provided encoding. Final text anomalies are corrected
   with ``ftfy``.

   :param raw_bytes: Input byte content to normalize.
   :param fallback_encoding: Encoding used if detection fails. Defaults to "utf-8".

   :returns: Normalized text as ``str``.

   :raises ValueError: If ``raw_bytes`` is ``None``.


.. py:function:: normalize_text(text, fallback_encoding = 'utf-8')

   Normalize arbitrary text or bytes to a cleaned ``str``.

   :param text: Text as ``str`` or ``bytes``.
   :param fallback_encoding: Encoding used when ``text`` is ``bytes`` and detection
                             fails. Defaults to "utf-8".

   :returns: Normalized text as ``str``.

   :raises ValueError: If ``text`` is ``None``.
