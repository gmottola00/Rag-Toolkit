.. py:function:: parse_docx(path)

   Parse paragraphs and tables from a DOCX file.

   :param path: Path to the DOCX file.

   :returns: Dictionary containing ``blocks`` (paragraphs/headings) and ``tables``.

   :raises FileNotFoundError: If the file does not exist.
   :raises ValueError: If the path is not a file.
   :raises RuntimeError: If python-docx cannot open the document.
