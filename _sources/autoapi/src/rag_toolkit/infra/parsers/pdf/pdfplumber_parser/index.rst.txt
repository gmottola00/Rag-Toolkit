.. py:function:: extract_pdf_layout(path)

   Extract text layout information from a PDF file using PyMuPDF.

   :param path: Path to the PDF file.

   :returns: List of pages with extracted blocks (text, bbox, font info).

   :raises FileNotFoundError: If the file does not exist.
   :raises ValueError: If the path is not a file.
   :raises RuntimeError: If PyMuPDF fails to read the document.


.. py:function:: parse_pdf(path, detect_headings = True, detect_tables = True)

   Parse a PDF file and optionally tag headings and tables.

   :param path: Path to the PDF file.
   :param detect_headings: Whether to classify heading-like blocks. Defaults to True.
   :param detect_tables: Whether to detect tables via pdfplumber and heuristics. Defaults to True.

   :returns: A list of page dictionaries, each containing enriched blocks.
