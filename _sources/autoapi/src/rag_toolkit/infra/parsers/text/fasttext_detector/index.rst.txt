.. py:function:: detect_language(text, max_chars = 5000, model_path = None)

   Detect the primary language of the given text.

   :param text: Input text.
   :param max_chars: Maximum characters to consider for detection. Defaults to 5000.
   :param model_path: Optional path to a fastText model. Defaults to ``models/lid.176.bin``.

   :returns: A lowercase ISO-639-1 code (e.g., ``"it"``, ``"en"``) or ``"unknown"``.

   :raises RuntimeError: If fastText is requested but the model file is missing and
       no alternative detector is available.
