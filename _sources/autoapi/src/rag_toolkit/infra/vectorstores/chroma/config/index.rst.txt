.. py:class:: ChromaConfig

   ChromaDB connection configuration.

   :param path: Path for persistent storage (None for in-memory)
   :param host: Remote ChromaDB server host (optional)
   :param port: Remote ChromaDB server port (optional)
   :param ssl: Use SSL for remote connection
   :param headers: Additional headers for remote connection
   :param tenant: Multi-tenancy tenant name (default: "default_tenant")
   :param database: Multi-tenancy database name (default: "default_database")
