src.rag_toolkit.infra.vectorstores.factory¶

Factory for creating vector store services and index services.

This module provides production-ready factory functions for instantiating vector stores with sensible defaults while allowing configuration overrides.

Create a configured MilvusService instance.

Parameters:

  • uri (Optional[str]) – Milvus connection URI (defaults to MILVUS_URI env var or localhost:19530).

  • user (Optional[str]) – Username for authentication (defaults to MILVUS_USER env var).

  • password (Optional[str]) – Password for authentication (defaults to MILVUS_PASSWORD env var).

  • db_name (Optional[str]) – Database name (defaults to MILVUS_DB env var or “default”).

  • secure (bool) – Use TLS encryption.

  • timeout (Optional[float]) – Connection timeout in seconds.

  • alias (str) – Connection alias for management.

Returns:

Configured MilvusService instance.

Return type:

rag_toolkit.infra.vectorstores.milvus.service.MilvusService

Example

```python from rag_toolkit.infra.vectorstores.factory import create_milvus_service

# Use environment variables service = create_milvus_service()

# Override specific settings service = create_milvus_service(

uri=”http://localhost:19530”, db_name=”production”

Create a configured IndexService with Milvus backend.

Parameters:

  • embedding_dim (int) – Dimensionality of embeddings.

  • embed_fn (Callable[[List[str]], List[List[float]]]) – Function to generate embeddings from text.

  • collection_name (Optional[str]) – Collection name (defaults to MILVUS_COLLECTION env var or “tender_chunks”).

  • metric_type (Optional[str]) – Distance metric (defaults to MILVUS_METRIC env var or “IP”).

  • index_type (Optional[str]) – Index type (defaults to MILVUS_INDEX_TYPE env var or “HNSW”).

  • vector_store (Optional[rag_toolkit.infra.vectorstores.milvus.service.MilvusService]) – Optional pre-configured vector store (creates new if None).

Returns:

Configured IndexService instance.

Return type:

rag_toolkit.core.index.service.IndexService

Example

```python from rag_toolkit.infra.vectorstores.factory import create_index_service from rag_toolkit.core.embedding import EmbeddingClient

# Create embedding client embed_client = EmbeddingClient(…)

# Create index service index_service = create_index_service(

embedding_dim=1536, embed_fn=lambda texts: embed_client.embed_batch(texts), collection_name=”my_collection”
