Running Examples¶

All examples are available in the examples/ directory:

# Clone the repository
git clone https://github.com/gmottola00/rag-toolkit.git
cd rag-toolkit

# Install with examples dependencies
pip install -e ".[all]"

# Run an example
python examples/basic_rag.py
